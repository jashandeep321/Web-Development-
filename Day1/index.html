Know More

HTML

The standard language used to structure and display content on the web using elements like headings, paragraphs, links, and images.

CSS

A styling language that controls the appearance and layout of HTML elements, including colors, fonts, spacing, and responsiveness.

Flexbox

A CSS layout model that makes it easy to align, distribute, and order items within a container along a single axis (row or column).

Grid

A powerful 2D layout system in CSS that enables the creation of complex, responsive web layouts with rows and columns.

Responsive

A web design approach that ensures websites look and function well across all devices and screen sizes, from mobile phones to desktops.

Media Query

CSS rules that apply styles based on screen size, resolution, or device type, enabling responsive design for different devices.
